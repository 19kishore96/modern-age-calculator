# 🎂 Modern Age Calculator

A modern **Age Calculator Web App** built with **HTML, CSS, and JavaScript**.  
It calculates your exact age in **years, months, and days** based on your date of birth, with proper input validation, leap year handling, and a gorgeous *glassmorphism UI design*. 💎✨  
Perfect for front-end beginners who are learning DOM manipulation and responsive UI design. 🚀

---

## 📦 Requirements

This project uses **plain HTML, CSS, and JavaScript**, so you only need a browser — no libraries or frameworks needed.

---

## 🧠 How It Works

1. User enters their **birth date (day, month, year)**  
2. JavaScript calculates the difference between the entered date and today’s date  
3. It handles leap years and date/month adjustments  
4. The result is displayed in years, months, and days

> All calculations are done client-side — no backend needed!

---

## ⚙️ Browser Compatibility

The app works smoothly in all modern browsers such as:

- Chrome
- Firefox
- Microsoft Edge
- Safari

If the blur effect doesn’t appear, ensure your browser supports **backdrop-filter**:

```css
backdrop-filter: blur(15px);
-webkit-backdrop-filter: blur(15px);
```
